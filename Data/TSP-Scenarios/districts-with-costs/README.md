# TSP-Scenarios

The files at this folder should have the following format:

```json
{
    "districts": [
        {
	    "average-cost": "double",
	    "list-blocks": ["int"],
	    "scenarios-cost": ["double"],
	    "scenarios-nbCustomers": ["int"]
	}
    ], 
    
    
    "metadata": {         
	 "TIME_TO_GENERATE": "float"
    }
}
```
